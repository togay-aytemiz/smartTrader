# SmartTrader Design System

## Font
**Plus Jakarta Sans** (Google Fonts) — weights: 300–800

## Color Palette
| Token | Hex | Use |
|-------|-----|-----|
| `--bg-base` | `#0c0c0e` | Page background |
| `--bg-surface` | `#141416` | Sunken areas |
| `--bg-card` | `#1a1a1e` | Cards |
| `--accent` | `#7c3aed` | Primary action |
| `--green` | `#22c55e` | Positive / bullish |
| `--red` | `#ef4444` | Negative / bearish |
| `--yellow` | `#f59e0b` | Warnings / neutral |

## Spacing
4px grid: `--space-1` (4px) through `--space-12` (48px)

## Radius
- Cards: `--radius-xl` (18px)
- Inputs: `--radius-md` (10px)
- Pills/badges: `--radius-pill` (999px)

## Files
- `tokens.css` — CSS custom properties (design tokens)
- All tokens imported via `index.css`
